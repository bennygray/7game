## 5-Why 根因链

```
为什么 BLOCK 穿透了？→ 判定标签与统计矛盾
为什么矛盾未被发现？→ 无校验规则
为什么修复后未重审？→ 协议是声明式的，无结构性防护
为什么签章也没拦住？→ 签章清单无重审文件完整性检查
根因：流程可靠性依赖执行者自觉性，缺乏机械性防护
```
