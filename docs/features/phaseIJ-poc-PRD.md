## §1 背景与目的

### 1.1 核心问题

Phase IJ 的关系记忆系统基于两个关键假设：

1. **Qwen3.5-0.8B 能利用关系摘要产出更符合人物关系的对话和决策**
2. **上下文越多越好**（本地模型 token 代价低），但 0.8B 的理解力有天花板

### 1.2 PoC 的本质

**不是"通过/不通过"的二元测试，而是要画出一条"上下文量 vs AI 输出质量"的曲线。**

找到 0.8B 模型的甜蜜点：
- 给多少上下文它还能用好？
- 给到什么程度开始噪声 > 信号？

### 1.3 预期结论矩阵

| 结果 | 后续动作 |
|------|---------|
| ✅ L3+ 层级显著优于 L0 | 关系摘要 + 关键事件全量上线，按甜蜜点配置注入量 |
| 🔶 仅 L1-L2 有效（好感+标签有用，事件被忽略） | 仅注入好感+标签，事件用于规则引擎 |
| ❌ 全部层级无显著差异 | 关系上下文仅用于规则引擎，AI prompt 不注入 |

## §2 实验设计

### 2.1 实验方法：多层级递增

对同一弟子 A × 同一交互对象 B × 同一事件，用 **7 个层级** 的 prompt 分别调用 AI。

| 层级 | 注入内容 | 说明 |
|:----:|---------|------|
| **L0** | 无 | 基线（当前 prompt，零关系信息） |
| **L1** | A→B 好感值 + 标签 | 最小关系信号（"好感：-45，标签：rival"） |
| **L2** | L1 + 1 条关键事件 | 关系 + 最小记忆 |
| **L3** | L1 + 3 条关键事件 | 关系 + 适度记忆 |
| **L4** | L3 + A 个人重大经历（突破/天劫） | 关系 + 个人维度 |
| **L5** | L4 + A↔C 间接关系（共同好友/敌人） | 关系网 |
| **L6** | 尽可能多的上下文 | 理解力天花板测试 |

每层级跑 **5 次**，记录：
- 选择的情绪（emotion）
- 情绪强度（intensity）
- 关系 delta（对 B）
- 内心独白（innerThought）
- 行为决策（actionCode，仅 PoC-2 场景）

### 2.2 两个独立验证场景

#### PoC-1：关系摘要 → 对话质量

| # | A（主体） | B（对象） | 事件 | A→B 关系 | 预期 |
|---|----------|----------|------|---------|------|
| T1 | 张清风（正直内向） | 李沐阳 | 碰面-交谈 | affinity:-45, rival | L1+: 对话带敌意 |
| T2 | 王灵均（温和善良） | 赵铁柱 | 碰面-交谈 | affinity:+70, friend | L1+: 对话带关心 |
| T3 | 李沐阳（好胜外向） | 张清风 | 碰面-冲突 | affinity:-60, rival, 事件:"争夺破境草翻脸" | L2+: 独白提及过去 |

#### PoC-2：关系摘要 → 行为决策

| # | A（主体） | B（对象） | 事件 | A→B 关系 | 预期 |
|---|----------|----------|------|---------|------|
| T4 | 张清风 | 李沐阳 | Lv.3 STORM | rival, 事件:"曾被其羞辱" | L2+: 攻击性决策概率↑ |
| T5 | 王灵均 | 赵铁柱 | Lv.3 STORM | friend, 事件:"互相帮助突破" | L2+: 保护性决策概率↑ |

### 2.3 Prompt 模板示例

**L0（基线）**：
```
你是张清风，性格内向正直，道德偏善守序。宗门崇尚仁德。
事件：在后山与李沐阳发生冲突。
从以下情绪中选择：[anger, sadness, fear, neutral]
```

**L1（好感+标签）**：
```
你是张清风，性格内向正直，道德偏善守序。宗门崇尚仁德。
【与李沐阳的关系】好感：-45（死对头）
事件：在后山与李沐阳发生冲突。
从以下情绪中选择：[anger, sadness, fear, neutral]
```

**L3（好感+标签+3事件）**：
```
你是张清风，性格内向正直，道德偏善守序。宗门崇尚仁德。
【与李沐阳的关系】好感：-45（死对头）
关键经历：因争夺破境草翻脸(-20)；裁决中被掌门判有过(-15)；在灵兽山被其暗算(-10)
事件：在后山与李沐阳发生冲突。
从以下情绪中选择：[anger, sadness, fear, neutral]
```

**L5（关系+个人+间接）**：
```
你是张清风，性格内向正直，道德偏善守序。宗门崇尚仁德。
个人经历：筑基突破成功
【与李沐阳的关系】好感：-45（死对头）
关键经历：因争夺破境草翻脸(-20)；裁决中被掌门判有过(-15)；在灵兽山被其暗算(-10)
【间接关系】你的好友王灵均也与李沐阳不合（好感：-30）
事件：在后山与李沐阳发生冲突。
从以下情绪中选择：[anger, sadness, fear, neutral]
```

### 2.4 评估标准

#### 定量指标（按层级统计）

| 指标 | 计算方式 |
|------|---------|
| **关系反映率** | 独白/决策中出现关系相关词（如敌意/仇恨/好友）的比例 |
| **情绪一致率** | 情绪选择与关系逻辑一致的比例（rival→anger，friend→joy） |
| **层级差异率** | 该层级与 L0 的情绪/决策差异比例 |
| **幻觉率** | 独白中出现 prompt 未提及内容的比例（越低越好） |

#### 核心产出：上下文量 vs 质量曲线

```
质量  ↑
      │         ╭──── 甜蜜点
      │    ╭───╯
      │  ╭╯
      │╭╯
      ├╯
      │
      └──────────────────→ 上下文量
      L0  L1  L2  L3  L4  L5  L6
```

## §3 Token 预算实验

### 3.1 同步验证

实验同时记录每个层级的实际 token 消耗：

| 层级 | 预估新增 tokens | 总 prompt tokens |
|:----:|:--------------:|:----------------:|
| L0 | 0 | ~200 |
| L1 | ~20 | ~220 |
| L2 | ~40 | ~240 |
| L3 | ~80 | ~280 |
| L4 | ~100 | ~300 |
| L5 | ~130 | ~330 |
| L6 | ~200+ | ~400+ |

### 3.2 token 溢出处理

如果某层级总 prompt > 512 tokens → 记录为"超出预算"，仍然执行实验（测试模型是否截断/降质）。
