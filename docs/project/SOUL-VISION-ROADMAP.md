## Phase 路线图

### Phase F — 灵魂闭环 `[决策层修复]`

> **一句话**：让特性、情绪、关系第一次真正影响弟子的行为选择。
> **价值**：从「有灵魂数据的傀儡」变成「行为可被解释的活人」。
> **前置**：Phase E ✅

**交付物**：

| # | 内容 | 说明 |
|---|------|------|
| F1 | 特性→行为权重叠加 | `getPersonalityWeights` 读取弟子 traits 中 `behavior-weight` 效果并叠加到权重向量 |
| F2 | 关系标签效果接入 | friend/rival 的 delta 倍率（×1.2/×0.8）在 soul-engine 中生效 |
| F3 | 短期情绪状态 | 弟子维护 `lastEmotion` + `emotionDecayTimer`，强烈情绪暂时偏移行为权重 |
| F4 | 关系标签→行为权重 | friend→合作行为(farm/bounty)×1.3, admirer→meditate×1.1 |

**验收标准**：同一事件下，持有「胆魄如虹」的弟子明显更多选择 explore，持有「丹道痴迷」的弟子明显更多选择 alchemy（Monte Carlo N=1000 行为分布差异 ≥15%）。

---

### Phase G — AI 觉醒 `[感知层接通]`

> **一句话**：将 AI 推理从死代码变为活系统，让弟子的内心独白和情绪由 AI 生成。
> **价值**：从「模板化反应」变成「每次都不同的、有个性的反应」。
> **前置**：Phase F ✅

**交付物**：

| # | 内容 | 说明 |
|---|------|------|
| G1 | soul-prompt-builder 接入 llm-adapter | processSoulEvent 的 AI 路径打通，fallback 作为兜底 |
| G2 | 对话系统读取关系上下文 | prompt-builder 植入 affinity/tags，好友和仇人说话语气不同 |
| G3 | 特性 aiHint 注入对话 prompt | 弟子的对话和内心独白反映其特性 |
| G4 | async 竞态处理 | processSoulEvent 改回 async，选择方案 A 或 B 处理 pipeline 竞态 |

**验收标准**：AI 在线时，同一事件对「善妒」弟子和「忠义」弟子产生明显不同的内心独白和情绪选择。

---

### Phase H — 弟子画像 `[表现层搭建]`

> **一句话**：让玩家能直观地看到、理解、感受每个弟子的灵魂。
> **价值**：灵魂系统的价值被玩家感知，从「后台数据」变成「前台体验」。
> **前置**：Phase F ✅（可与 G 并行）

**交付物**：

| # | 内容 | 说明 |
|---|------|------|
| H1 | 弟子档案面板 | 显示特性列表、道德标签（至善/冷酷等）、当前情绪 |
| H2 | 关系面板 | 弟子间 affinity 可视化 + 关系标签显示 |
| H3 | 灵魂事件高亮 | 关键灵魂事件（标签变化、后天特性获得）在 MUD 日志中置顶/高亮 |
| H4 | 弟子状态指示 | MUD 面板中弟子名旁显示当前情绪 emoji |

**验收标准**：玩家能在 UI 中主动查看任一弟子的特性、道德、关系数据，无需翻阅日志。

---

### Phase I — 活世界 `[社交层+世界事件]`

> **一句话**：弟子之间产生超越日常行为的社交事件，世界不再只有修炼循环。
> **价值**：从「各做各事」变成「有恩怨、有故事」。
> **前置**：Phase F ✅, Phase G ✅

**交付物**：

| # | 内容 | 说明 |
|---|------|------|
| I1 | 高级关系标签 | mentor（恩师）/admirer（崇拜）/grudge（心结）的触发和效果 |
| I2 | 社交事件系统 | 争吵、和好、结拜等事件，由 affinity/emotion 阈值触发 |
| I3 | 道德趋同 | 亲密弟子长期互相影响道德值（R-E14 完整版） |
| I4 | 天劫系统 | 炼气→筑基的关卡，突破时全宗弟子反应 |
| I5 | 悬赏任务 | 弟子外出历练的内容充实 |

**验收标准**：运行 30 分钟内，至少自然发生 1 次社交事件（结拜/争吵/恩师认定）。

---

### Phase J — 涌现与深度 `[高阶智能]`

> **一句话**：NPC 拥有记忆、目标、自驱力，行为超越规则预设，产生涌现式的故事。
> **价值**：从「有灵魂的棋子」变成「有故事的人」。
> **前置**：Phase I ✅

**交付物**：

| # | 内容 | 说明 |
|---|------|------|
| J1 | 短期记忆 | 弟子记住最近 N 次重要事件，影响对话和决策 |
| J2 | 长期记忆摘要 | AI 定期压缩事件为性格化记忆摘要 |
| J3 | 个人目标系统 | 弟子根据性格/处境自动生成欲望（想突破/想复仇/想隐居） |
| J4 | 目标驱动行为 | 目标改变行为权重优先级（想突破→疯狂修炼） |
| J5 | 叛逃/推举事件 | 极端关系+道德组合触发的剧烈事件 |

**验收标准**：弟子能自发说出「我记得上次张清风炼丹失败时冷嘲热讽，我不会忘记」这样基于记忆的对话。
