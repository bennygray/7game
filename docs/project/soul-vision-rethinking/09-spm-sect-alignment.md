## 1. 双轴设计：道风 × 门规

> 不用"善恶"——道没有纯粹的善恶。用"仁/霸"和"律/放"来表达宗门气质。

### 横轴 —— 道风 (Sect Ethos)：宗门对"道"的理解

| ← 仁 (-100) | 中 (0) | 霸 (+100) → |
|:---:|:---:|:---:|
| 行善济世、扶弱抑强 | 各取所需、不偏不倚 | 实力至上、弱肉强食 |

### 纵轴 —— 门规 (Sect Discipline)：宗门如何管理弟子

| ↑ 律 (+100) | 中 (0) | ↓ 放 (-100) |
|:---:|:---:|:---:|
| 清规戒律、犯者重罚 | 有规矩但不死板 | 各行其是、自负其责 |

### 四象限

```
                      律 (+100)
                       │
          仁义正宗 ┌───┼───┐ 铁血霸宗
         (武当式) │   │   │ (日月神教式)
         济世+戒律│   │   │ 霸道+铁律
                  │   │   │
   仁 (-100) ─────┼───┼───┼────── 霸 (+100)
                  │   │   │
          逍遥侠宗│   │   │ 修罗魔宗
         (丐帮式) │   │   │ (星宿派式)
         济世+自由└───┼───┘ 弱肉强食+放任
                       │
                      放 (-100)
```

| 象限 | 宗门气质 | 弟子日常 | AI 对话调性 |
|------|---------|---------|-----------|
| 仁+律 | **仁义正宗** | 互相扶持，日出论道，严禁私斗 | 正气凛然、师兄弟情深 |
| 仁+放 | **逍遥侠宗** | 性格各异但底线不害人，掌门佛系 | 洒脱随性、侠骨柔肠 |
| 霸+律 | **铁血霸宗** | 等级森严，弱者淘汰，但团结对外 | 冷厉高效、忠诚至上 |
| 霸+放 | **修罗魔宗** | 同门即对手，夺丹叛逃家常便饭 | 阴狠算计、强者为尊 |

> **没有"善"也没有"恶"**——仁义正宗可能因铲除邪道而屠灭无辜；修罗魔宗中也可能有不离不弃的生死之交。

## 3. 漂移规则（USER 定稿）

### 日常裁决：水滴石穿

| 行为 | 道风变化 | 门规变化 |
|------|:-------:|:-------:|
| 严惩弟子恶行 | -1 (偏仁) | +1 (偏律) |
| 包庇弟子恶行 | +1 (偏霸) | -1 (偏放) |
| 公正赏善 | -1 (偏仁) | +1 (偏律) |
| 偏袒强者 | +1 (偏霸) | -1 (偏放) |

**幅度极小（±1~2），几乎感知不到。** 数十次裁决才移动一个肉眼可见的刻度。

### Lv.4 世界事件：真正的转折

**只有天劫级的宗门命运抉择才会大幅改变门风（±10~20）：**

| 事件 | 仁道选择 | 霸道选择 |
|------|---------|---------|
| 被正道联盟追杀的邪修天才来投 | 拒绝收容（仁-0，律+2） | 收容并保护（霸+15，放+5） |
| 宗门存亡之际，魔道宗门提议结盟 | 宁死不屈（仁+10，律+10） | 为生存结盟（霸+15，放-5） |
| 弟子在外杀了正道弟子被发现 | 交出弟子受审（仁+10，律+15） | 包庇到底，灭口证人（霸+20，放-5） |
| 发现上古禁术，威力巨大但损人功德 | 封印禁术（仁+10） | 修炼禁术（霸+15） |

## 4. 宗门道风影响全局

### 4.1 事件池调性

| 道风倾向 | 更常刷的事件 | 更少刷的事件 |
|---------|-----------|-----------|
| 仁道 | 散修求助、除妖任务、正道结盟 | 暗杀、夺宝 |
| 霸道 | 叛逃投靠、地下交易、仇敌来袭 | 和平外交 |
| 律 | 弟子违规惩戒、门规审查 | 自发冲突 |
| 放 | 弟子间自发冲突、内斗 | 纪律性事件 |

### 4.2 AI Prompt 注入

宗门道风自动注入所有 AI 调用的系统 prompt：

- **仁义正宗**：*"你是仁义正宗的弟子。宗门崇尚济世扶弱，同门以礼相待。"*
- **修罗魔宗**：*"你是修罗魔宗的弟子。宗门崇尚实力至上，同门既是伙伴也是竞争者。"*

AI 自然产出完全不同调性的独白和对话，**无需额外代码逻辑**。

### 4.3 T2 幕后 NPC 态度

| 我方道风 | 仁道 NPC | 霸道 NPC | 中立 NPC |
|---------|:-------:|:-------:|:-------:|
| 仁道 | 亲近 | 轻视 | 友善 |
| 霸道 | 警惕 | 尊重/竞争 | 戒备 |

### 4.4 弟子与宗门的道风冲突

弟子个人 moral 与宗门道风差距过大 → 离心力增加：
- 善良弟子在霸道宗门 → 日渐不满 → 可能触发"弟子叛逃"Lv.4 事件
- 贪婪弟子在仁义宗门 → 私下搞事 → 可能触发"内鬼"Lv.3 事件

**这让弟子个体的道德系统和宗门系统产生了真正的互动闭环。**

## 5. 堕落与救赎叙事（渐进式）

道风跨过关键阈值时触发叙事事件链：

### 从仁走向霸（堕落）

```
道风从 -40 逐渐滑向 +...
  -20: [Lv.2] 弟子私下议论"掌门最近的手段是不是太狠了？"
    0: [Lv.3] 老资格弟子求见——"掌门，我们的道还是当初的道吗？"
  +20: [Lv.4] ⚡ 仁道联盟发来最后通牒；善良弟子开始动摇
  +40: 宗门气质彻底转为霸道，事件池全面切换
```

### 从霸走向仁（救赎）

```
道风从 +40 逐渐滑向 -...
  +20: [Lv.2] 有弟子开始反思"我们是不是太过了？"
    0: [Lv.3] 一位仁道散修来访——"听闻贵宗近来多行善举……"
  -20: [Lv.4] ⚡ 霸道弟子不满宗门"变软"，密谋夺权
  -40: 宗门洗白为仁道，但代价是流失一批霸道弟子
```

**这些转折只有在 Lv.4 世界事件中做出重大抉择才会发生。日常的小裁决只是水滴石穿的铺垫。**
